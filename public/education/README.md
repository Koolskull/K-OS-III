# Education tracks

Each subfolder is a track. Each track has a `track.yaml` manifest and numbered lesson files (`01-...md`, `02-...md`, …).

For the index and the project's framing of these tracks, see [`EDUCATION.md`](../../EDUCATION.md) at the repo root.
For the system spec, see [`docs/KNOWLEDGE_CORPUS.md`](../../docs/KNOWLEDGE_CORPUS.md).
